# yieldwatch-prometheus-exporter

Yieldwatch Prometheus Exporter

## Install and run

```bash
$ virtualenv -p python3 venv
$ source venv/bin/activate
$ pip install -r requirements.txt
$ python main.py --debug --wallet 0x3Ab3...
```
